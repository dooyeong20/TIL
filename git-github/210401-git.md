# How to use branch ?

## Branch

- git checkout {branch_name}
- git switch {branch_name} 이렇게 바뀌었다

## Conflict

브랜치를 쳐서 작업을 하던 도중, 동시에 작업을 변경했을 경우, 충돌이 발생하게 된다.

### 예시

```python
<<<<<<< HEAD
# print function on main branch
for i in range(1, 10 + 1):
    if i % 5 == 0:
        print(f'{i} : hello, main !')
=======
# print function on stem branch
result = ['' if i % 3 == 0 else i for i in range(1, 10 + 1)]
print(result)
>>>>>>> stem

```

그리고 merge는 commti이 특별하게 미리 정해져 있다.

- 만약 Critical한 경우, 내역을 적어주면 된다.

## git flow

- git feature start

## git 삭제

- mv {file} {renamed}

  - 이렇게하면 추적을 못하고 따로따로 인식하게 됨

- git mv {file} {renamed}
  - 이렇게 해줘야 git이 파일 변경을 추적할 수 있음

## git Unstaging Undoing

- unstaging
  - "reset HEAD {filename}"
- undo
  - "restore {filename}"
  - or "checkout -- {filename}"
  - 최신 커밋으로 되돌아 감
